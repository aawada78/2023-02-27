Getting Started

Prerequisites

For the labs, you need the following software packages:

  • NodeJS in current LTS version
  • Angular CLI (npm i -g @angular/cli)
  • Visual Studio Code (free) or WebStorm/IntelliJ (commercial)
  • Git Client

Starter Kit

Please clone our starter kit for the exercises and install the needed npm packages:

git clone https://github.com/aawada78/2023-02-27.git
cd 2023-02-27
npm i
